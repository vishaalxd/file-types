Raster formats

jpg


acronym for Joint Photographic Experts.
Typically used for phtographs and complex images
Can be scaled to around 130% before you can see the loss in quality



Where to Use

Used for product shots, portraits/head shots and other images where color variances are important.


png


acronym for Portable Network Graphics.
Created as an atlernative to the gif file format.
Supports transparency.



Where to Use

Used for images involving text, charts, screen shots and detailed illustrations.


gif


acronym for Graphics interchange Format.
Reduces the number of colors in an image to 256.
Supports transparency.
png's and gif's are very similar but png does everything a little better.



Where to Use

Used for images involving text, charts, screen shots and detailed illustrations.


psd


Adobe Phtoshop Extension.
Layered working file that is editable in Adobe Photosop.



Where to Use

Used for editing in Adobe Photoshop.


Vector formats


acronym for encapulated postscript file.
Typically used for logos and illustrations.
Can be scaled to any size without losing quality.
Supports transparency



Where to Use

Printed collateral - especially large scale printing(banners,posters,etc)


Ai


Adobe illustrator Extension.
Layered working file that is editable in Adobe Photosop.



Where to Use

Used for editing in Adobe Illustrator.


pdf


acronym for Portable Document Format.
In some cases, vector elements such as logos and illustrations as well as text can be pulled from pdf files.
Essential digital version of a hard copy print out.



Where to Use

It is generally used for emailing documents.
